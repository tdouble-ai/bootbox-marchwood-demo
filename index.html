About this demo

Search runs on keywords first — type steel and you'll find it instantly. If keyword search comes up empty, this demo tries a limited semantic fallback, so structural material can still surface steel and concrete records. It's capped at a few hundred lookups a day since this is a free public demo — the real engagement runs semantic search with no caps, built into whatever system already holds your data. There's also a new Family Grid view — same data, sorted into groups instead of floating free, if the web gets busy.

Bootbox is my business-context layer, built for TDoubleAI clients. This public version is a snapshot so anyone can click through without an account. For actual clients it runs live — linked straight to Notion or whatever secure system already holds your data.
